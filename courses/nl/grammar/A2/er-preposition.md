# er + preposition / waar + preposition

**CEFR level:** A2  
**Introduced in lesson:** 32  
**Depends on:** [er](er.md) | [Prepositions of place](../A1/prepositions-of-place.md)

---

## Explanation

When a preposition refers back to a **non-human noun or idea**, Dutch does not use *preposition + het/ze*. Instead it forms a **pronominal adverb** by combining **er** (or **waar** for questions and relative clauses) with the preposition.

```
Ik denk over het probleem.  →  Ik denk erover.      (I think about it)
Waar denk je over?           →  Waar denk je over?   (What are you thinking about?)
```

---

## Forming *er* + preposition

Combine **er** directly with the preposition — usually written as one word:

| Preposition | er + prep | Example |
|------------|----------|---------|
| over | **erover** | Ik praat erover. |
| mee | **ermee** | Wat doe je ermee? |
| op | **erop** | De boeken liggen erop. |
| in | **erin** | Doe het erin. |
| aan | **eraan** | Ik denk eraan. |
| voor | **ervoor** | Ze werkt ervoor. |
| uit | **eruit** | Haal het eruit. |
| naast | **ernaast** | De fiets staat ernaast. |
| achter | **erachter** | Wat zit erachter? |
| tussen | **ertussen** | Stop het ertussen. |
| van | **ervan** | Ik hou ervan. |
| door | **erdoor** | Laat het erdoor. |
| bij | **erbij** | Kom erbij zitten. |
| om | **erom** | Ik lach erom. |
| tegen | **ertegen** | Ik ben ertegen. |

---

## Splitting *er* from the preposition

*Er* can be split away from the preposition, especially in speech, with other material in between. Both orders are correct; split forms are common:

| Together | Split |
|---------|------|
| Ik denk **erover**. | Ik denk **er** veel **over**. |
| Ze praat **ermee**. | Ze praat **er** graag **mee**. |
| Hij houdt **ervan**. | Hij houdt **er** erg **van**. |

---

## Forming *waar* + preposition (questions and relative clauses)

For **questions** and **relative clauses** about things, use **waar** + preposition instead of a question word + preposition:

| *er* form | *waar* form | Use |
|----------|------------|-----|
| erover | **waarover** | Waarover praat je? |
| ermee | **waarmee** | Waarmee schrijf je? |
| erop | **waarop** | Waarop staat het? |
| erin | **waarin** | Waarin zit het? |
| ervan | **waarvan** | Waarvan hou je? |
| ervoor | **waarvoor** | Waarvoor gebruik je dat? |
| eruit | **waaruit** | Waaruit bestaat het? |

Like *er + preposition*, *waar + preposition* can also be split:

> *Waar praat je over?* = *Waarover praat je?*  
> *Waar schrijf je mee?* = *Waarmee schrijf je?*

---

## People vs things

| Referent | Use | Example |
|----------|-----|---------|
| **Thing / idea** | er/waar + preposition | *Ik denk erover.* / *Waarover denk je?* |
| **Person** | preposition + pronoun / wie | *Ik denk aan hem.* / *Aan wie denk je?* |

This distinction is non-negotiable. Never use *er* when referring to a person:

*Ik praat erover met haar.* ✓ (*erover* for the topic; *met haar* for the person)  
*Ik praat ermee.* ✓ (talking with it — a thing)  
*Ik praat met hem.* ✓ (talking with him — a person)

---

## *waar* + preposition in relative clauses

When a preposition refers to the antecedent in a relative clause, use *waar + preposition*:

| Dutch | English |
|-------|---------|
| Het boek **waarover** ik praat. | The book I am talking about. |
| De stoel **waarop** hij zit. | The chair on which he sits. |
| Het gereedschap **waarmee** ik werk. | The tool with which I work. |
| De reden **waarom** ik kom. | The reason why I come. |

---

## Examples

| Dutch | English |
|-------|---------|
| Ik hou **ervan** om te lezen. | I love reading. *(lit. I like it — reading)* |
| **Waarmee** schrijf jij — met een pen of een potlood? | What do you write with — a pen or a pencil? |
| Hij is **ertegen** — hij wil het niet. | He is against it — he doesn't want it. |
| **Waarover** gaat het gesprek? | What is the conversation about? |
| Ze is trots **erop**. / Ze is **er** trots **op**. | She is proud of it. |
| De tas **waarin** ze haar sleutels stopte. | The bag in which she put her keys. |

---

## Common mistakes

**1. Using *het* after a preposition for a thing**

*Ik denk over **het**.* ✗ → *Ik denk **erover**.* ✓

**2. Using *er* for a person**

*Ik bel **erop**.* ✗ (for a person) → *Ik bel **hem/haar**.* ✓

**3. Using *wat* instead of *waar* in questions**

*Met **wat** schrijf je?* ✗ → *Met **wie** schrijf je?* (person) / ***Waarmee** schrijf je?* (thing) ✓

**4. Splitting mandatory vs optional**

Both split and unsplit forms are correct. In subordinate clauses, split forms are slightly more natural: *Ik weet dat hij **er** over **nadenkt**.*

---

## Quick reference

```
Thing:    er  + preposition  →  erover / ermee / erop / erin / eraan …
Question: waar + preposition →  waarover / waarmee / waarop / waarin …
Person:   preposition + pronoun/wie  (met hem, aan wie)

Can split:  er … over  =  erover  (both correct)
Relative:   het boek waarover / de stoel waarop / de reden waarom
```
