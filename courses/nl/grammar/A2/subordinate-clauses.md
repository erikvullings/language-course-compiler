# Subordinate clauses

**CEFR level:** A2  
**Introduced in lesson:** 21  
**Depends on:** [Main clause word order](../A1/word-order.md)

---

## Explanation

A **subordinate clause** (*bijzin*) is a clause introduced by a conjunction that cannot stand alone as a sentence. The crucial difference from a main clause is word order: in a subordinate clause, **the finite verb moves to the very end**.

```
Main clause:    Ik werk thuis.          (verb in position 2)
Subordinate:    … omdat ik thuis werk.  (verb at the end)
```

---

## Common subordinating conjunctions

| Dutch | English | Type |
|-------|---------|------|
| **dat** | that | content |
| **omdat** | because | reason |
| **als** | if / when (present/future) | condition / time |
| **wanneer** | when | time |
| **terwijl** | while | time |
| **hoewel** / **alhoewel** | although | concession |
| **of** | whether / if (indirect question) | content |
| **totdat** | until | time |
| **nadat** | after | time |
| **voordat** | before | time |
| **zodra** | as soon as | time |
| **toen** | when (single past event) | time |
| **zodat** | so that | result |
| **tenzij** | unless | condition |

---

## Word order in subordinate clauses

### Single finite verb

The finite verb moves to the end:

| Main clause | Subordinate clause |
|-------------|-------------------|
| Ik werk thuis. | Hij zegt dat ik thuis **werk**. |
| Ze is moe. | Ik blijf thuis omdat ze moe **is**. |
| Hij kan morgen komen. | Ik hoop dat hij morgen kan **komen**. |

### With a modal verb or *gaan*

Both verbs go to the end, with the finite verb last (infinitive before finite):

> *Ik denk dat ze morgen gaan **vertrekken**.* — infinitive *vertrekken* before finite *gaan*  
> *Hij zegt dat ik het kan **doen**.* — infinitive *doen* before finite *kan*

### With perfect tense

Auxiliary and past participle both go to the end. The past participle comes before the auxiliary:

> *Ik weet dat hij gisteren **gewerkt heeft**.*  
> or: *Ik weet dat hij gisteren heeft **gewerkt**.* — both orders are acceptable

---

## The subordinate clause as the first element

A subordinate clause can open a sentence. When it does, the main clause immediately follows — and the main clause still requires inversion (V2):

```
[Subordinate clause]  ,  [Finite verb]  [Subject]  [Rest]
 Omdat ik ziek was    ,   bleef          ik         thuis.
 Because I was ill         stayed         I          home.
```

| Dutch | English |
|-------|---------|
| **Omdat het regent**, blijf ik binnen. | Because it is raining, I stay inside. |
| **Als je tijd hebt**, kom dan even langs. | If you have time, come by for a bit. |
| **Hoewel hij moe was**, werkte hij door. | Although he was tired, he kept working. |

---

## *dat* after verbs of saying, thinking, knowing

The conjunction *dat* (that) links a main clause to a content clause. In English *that* is often omitted; in Dutch *dat* is usually kept:

| Dutch | English |
|-------|---------|
| Ik **denk** dat ze morgen komt. | I think (that) she comes tomorrow. |
| Hij **weet** dat het moeilijk is. | He knows (that) it is difficult. |
| Ze **zegt** dat ze geen tijd heeft. | She says (that) she has no time. |
| Ik **hoop** dat het goed gaat. | I hope (that) things go well. |

---

## *of* for indirect yes/no questions

When an indirect question has a yes/no form, use *of* (whether):

| Direct question | Indirect form |
|----------------|--------------|
| Komt hij? | Ik vraag me af **of** hij komt. |
| Heb je tijd? | Ze vraagt **of** je tijd hebt. |
| Is het duur? | Ik weet niet **of** het duur is. |

---

## Examples

| Dutch | English |
|-------|---------|
| Hij bleef thuis **omdat** hij ziek **was**. | He stayed home because he was ill. |
| Ik weet niet **of** ze **komt**. | I don't know whether she's coming. |
| **Als** je honger **hebt**, kun je iets eten. | If you're hungry, you can eat something. |
| Ze wacht **totdat** de bus **aankomt**. | She waits until the bus arrives. |
| Ik geloof dat hij gelijk **heeft**. | I believe he is right. |
| **Hoewel** het koud **was**, gingen we fietsen. | Although it was cold, we went cycling. |

---

## Common mistakes

**1. Keeping the verb in position 2 inside a subordinate clause**

*Ik denk dat hij **komt** morgen.* ✗ → *Ik denk dat hij morgen **komt**.* ✓

**2. Wrong conjunction order with modal + infinitive**

*dat hij kan morgen komen* ✗ → *dat hij morgen **komen kan*** — both orders occur in speech, but infinitive-before-finite is standard.

**3. Starting the second main clause without inversion**

*Omdat het regent, ik blijf thuis.* ✗ → *Omdat het regent, **blijf ik** thuis.* ✓

**4. Using *als* for a single past event**

*als ik klein was* ✗ (as a single past moment) → *toen ik klein **was*** ✓  
(*als* = if/when (habitual/future); *toen* = when (single past event))

---

## Quick reference

```
Subordinating conjunctions → verb to the end
  single verb:       … omdat hij ziek is
  modal + inf.:      … omdat hij kan komen
  perfect:           … omdat hij gewerkt heeft

Fronted subordinate clause → inversion in main clause
  Omdat het regent, blijf ik thuis.

dat  = that (content)
of   = whether (indirect yes/no question)
als  = if / when (habitual or future)
toen = when (single past event)
```
