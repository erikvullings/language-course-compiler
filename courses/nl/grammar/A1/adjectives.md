# Attributive adjectives

**CEFR level:** A1  
**Introduced in lesson:** 14  
**Depends on:** [Definite articles](articles.md) | [Indefinite article](indefinite-article.md)

---

## Explanation

An **attributive adjective** modifies a noun directly: *a big house*, *the red car*. In Dutch, attributive adjectives receive an **-e** ending in most situations. There is one significant exception.

A **predicative adjective** follows a linking verb: *the house is big*. Predicative adjectives in Dutch receive **no ending at all**:

| Predicative (no ending) | Attributive (may add -e) |
|------------------------|--------------------------|
| Het huis is **groot**. | het **grote** huis |
| De fiets is **rood**. | de **rode** fiets |

This lesson covers the attributive case only.

---

## The rule and the one exception

### The rule: add **-e**

Add *-e* to the adjective when it comes before a noun, in almost all cases:

| Situation | Example | English |
|-----------|---------|---------|
| Definite *de*-word | de **grote** fiets | the big bicycle |
| Definite *het*-word | het **grote** huis | the big house |
| Indefinite *de*-word | een **grote** fiets | a big bicycle |
| **Plural** (always *de*) | de **grote** fietsen | the big bicycles |

### The exception: **no -e** with *een* + singular *het*-word

When the adjective stands before **een** (indefinite) + a singular **het**-word, the *-e* is **dropped**:

| Situation | Example | English |
|-----------|---------|---------|
| Indefinite *het*-word, singular | een **groot** huis | a big house |
| Indefinite *het*-word, singular | een **mooi** kind | a beautiful child |
| Indefinite *het*-word, singular | een **leuk** feest | a fun party |

> Memory hook: **"bare een + bare het-word → bare adjective"**  
> As soon as you add *het* (the) or make it plural, the *-e* comes back.

## Summary table

| | *de*-word | *het*-word |
|--|-----------|-----------|
| **de / het** (definite) | de groot**e** fiets | het groot**e** huis |
| **een** (indefinite) | een groot**e** fiets | een groot huis ← **no -e** |
| **plural** (*de*) | de groot**e** fietsen | de groot**e** huizen |
| **geen / dit / dat / mijn …** | groot**e** | groot**e** |

---

## Spelling adjustments

The same spelling rules that apply to verb stems apply to adjective endings:

| Base adjective | + -e | Note |
|---------------|------|------|
| groot | grote | double vowel → single before -e |
| rood | rode | *oo* → *o* before -e |
| oud | oude | |
| lief | lieve | *-f* → *-v-* between vowels |
| dik | dikke | short vowel + single consonant → double consonant |

## Predicative vs attributive at a glance

| | Dutch | English |
|--|-------|---------|
| Predicative | Het huis is **groot**. | The house is big. |
| Attributive (definite) | het **grote** huis | the big house |
| Attributive (*een* + het) | een **groot** huis | a big house |

---

## Examples

| Dutch | English |
|-------|---------|
| Ik heb een **mooi**e fiets. | I have a beautiful bicycle. |
| Hij woont in een **groot** huis. | He lives in a big house. |
| De **kleine** kinderen slapen. | The small children sleep. |
| Ze draagt een **rode** jas. | She wears a red coat. |
| Dit is een **leuk** boek. | This is a fun book. |
| De **oude** man loopt langzaam. | The old man walks slowly. |
| Een **dikke** kat zit op het dak. | A fat cat sits on the roof. |

---

## Common mistakes

**1. Always adding -e regardless of context**

*een grote huis* ✗ → *een groot huis* ✓ (*huis* is a het-word with *een*)

**2. Never adding -e**

*de groot fiets* ✗ → *de grote fiets* ✓

**3. Forgetting spelling changes**

*een roode fiets* ✗ → *een rode fiets* ✓  
*een dikee kat* ✗ → *een dikke kat* ✓

**4. Adding -e to predicative adjectives**

*De fiets is grote.* ✗ → *De fiets is groot.* ✓

---

## Quick reference

```
Predicative (after zijn/worden):   no ending  →  groot
Attributive + de-word:             -e         →  grote
Attributive + het-word + de/het:   -e         →  grote
Attributive + een + het-word:      no ending  →  groot  ← exception!
Attributive plural:                -e         →  grote
```
