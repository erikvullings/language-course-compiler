# Modal verbs

**CEFR level:** A1  
**Introduced in lesson:** 16  
**Depends on:** [Present tense](present-tense.md) | [Main clause word order](word-order.md)

---

## Explanation

Modal verbs express ability, willingness, obligation, and permission. In Dutch they work the same way as in English: the modal verb is conjugated and a **second verb in the infinitive goes to the end of the clause**.

```
[Subject]  [Modal]  [Rest]  [Infinitive]
 Ik         kan      morgen   komen.
 I           can      tomorrow come.
```

The four core A1 modals are: **kunnen** (can), **willen** (want), **moeten** (must), **mogen** (may).

---

## Conjugation

### kunnen — can / to be able to

| Subject | Form |
|---------|------|
| ik | **kan** |
| jij/je | **kunt** / **kan** |
| u | **kunt** |
| hij/zij/het | **kan** |
| wij/we | **kunnen** |
| jullie | **kunnen** |
| zij/ze (pl.) | **kunnen** |

> *kunt* and *kan* are both correct for *jij*. *Kan jij…?* (inversion, no t-drop because the form is *kan*) is common in speech.

### willen — to want

| Subject | Form |
|---------|------|
| ik | **wil** |
| jij/je | **wilt** / **wil** |
| u | **wilt** |
| hij/zij/het | **wil** |
| wij/we | **willen** |
| jullie | **willen** |
| zij/ze (pl.) | **willen** |

### moeten — must / to have to

| Subject | Form |
|---------|------|
| ik | **moet** |
| jij/je | **moet** |
| u | **moet** |
| hij/zij/het | **moet** |
| wij/we | **moeten** |
| jullie | **moeten** |
| zij/ze (pl.) | **moeten** |

### mogen — may / to be allowed to

| Subject | Form |
|---------|------|
| ik | **mag** |
| jij/je | **mag** |
| u | **mag** |
| hij/zij/het | **mag** |
| wij/we | **mogen** |
| jullie | **mogen** |
| zij/ze (pl.) | **mogen** |

## Word order: infinitive to the end

The modal is conjugated and sits in position 2. The infinitive goes to the **very end** of the clause:

| Dutch | English |
|-------|---------|
| Ik **kan** morgen **komen**. | I can come tomorrow. |
| Ze **wil** een boek **lezen**. | She wants to read a book. |
| Jullie **moeten** nu **vertrekken**. | You (all) must leave now. |
| Hij **mag** hier niet **roken**. | He is not allowed to smoke here. |

With inversion (adverb or question starts the clause), the modal moves to position 2 and the infinitive still ends the clause:

> *Morgen **kan** ik komen.* — Tomorrow I can come.  
> ***Kan** jij komen?* — Can you come?

---

## Meanings and use

| Modal | Core meaning | Example |
|-------|-------------|---------|
| **kunnen** | ability / possibility | *Ik kan zwemmen.* — I can swim. |
| **willen** | desire / want | *Ik wil koffie drinken.* — I want to drink coffee. |
| **moeten** | obligation / necessity | *Ik moet werken.* — I must work. |
| **mogen** | permission | *Mag ik gaan?* — May I go? |

---

## Modal without infinitive (ellipsis)

When the meaning is clear from context, the infinitive can be omitted — especially with verbs of motion or *hebben*:

> *Ik moet naar huis.* — I have to go home. *(gaan* omitted)  
> *Ze wil een kopje koffie.* — She wants a cup of coffee. *(hebben* or *drinken* omitted)

---

## Examples

| Dutch | English |
|-------|---------|
| **Kan** jij dit **uitleggen**? | Can you explain this? |
| Ik **wil** graag **helpen**. | I would like to help. |
| We **moeten** de trein **halen**. | We have to catch the train. |
| **Mag** ik hier **zitten**? | May I sit here? |
| Hij **kan** heel goed **koken**. | He can cook very well. |
| Ze **wil** niet **gaan**. | She does not want to go. |
| Jullie **mogen** nu **gaan**. | You (all) may go now. |

## Common mistakes

**1. Putting the infinitive in position 2 (English pattern)**

*Ik kan komen morgen.* ✗ → *Ik kan morgen **komen**.* ✓

**2. Conjugating the infinitive**

*Ik wil gaat naar huis.* ✗ → *Ik wil naar huis gaan.* ✓

**3. Using *kunnen* for permission**

*Kan ik gaan?* can work colloquially, but *Mag ik gaan?* is the correct choice for permission.

**4. Forgetting *niet* goes before the infinitive**

*Ik kan komen niet.* ✗ → *Ik kan **niet** komen.* ✓

## Quick reference

```
kunnen  ik kan / jij kunt (kan) / hij kan / wij kunnen
willen  ik wil / jij wilt (wil) / hij wil / wij willen
moeten  ik moet / hij moet / wij moeten
mogen   ik mag / hij mag / wij mogen

Order:  [Subject] [Modal (pos.2)] [rest] [Infinitive (end)]
Niet:   before the infinitive → ik kan niet komen
```
