# Demonstratives (deze / die / dit / dat)

**CEFR level:** A1  
**Introduced in lesson:** 12  
**Depends on:** [Definite articles](articles.md)

---

## Explanation

Dutch demonstratives point to or identify a noun. There are four forms, determined by **two factors**:

1. **Gender** of the noun: *de*-word or *het*-word
2. **Distance**: near (*this/these*) or far (*that/those*)

| | **Near** (this / these) | **Far** (that / those) |
|--|------------------------|----------------------|
| **de**-word + plural | **deze** | **die** |
| **het**-word (singular only) | **dit** | **dat** |

> Plurals always behave like *de*-words and use **deze** (near) or **die** (far).

---

## The four forms

### deze — this (de-word or plural)

| Dutch | English |
|-------|---------|
| **Deze** fiets is van mij. | This bicycle is mine. |
| **Deze** kinderen zijn lief. | These children are sweet. |
| **Deze** tafel is te klein. | This table is too small. |

### die — that (de-word or plural)

| Dutch | English |
|-------|---------|
| **Die** auto is oud. | That car is old. |
| **Die** mensen kennen wij. | We know those people. |
| **Die** bomen zijn mooi. | Those trees are beautiful. |

### dit — this (het-word singular)

| Dutch | English |
|-------|---------|
| **Dit** huis is groot. | This house is big. |
| **Dit** boek is interessant. | This book is interesting. |
| **Dit** kind speelt buiten. | This child plays outside. |

### dat — that (het-word singular)

| Dutch | English |
|-------|---------|
| **Dat** huis staat leeg. | That house is empty. |
| **Dat** boek heb ik al gelezen. | I have already read that book. |
| **Dat** idee vind ik goed. | I think that idea is good. |

---

## Demonstratives as pronouns

The same four forms are used as **stand-alone pronouns** (replacing a noun entirely):

| Dutch | English |
|-------|---------|
| Wat is **dit**? | What is this? |
| Wie is **dat**? | Who is that? |
| **Dit** is mijn zus. | This is my sister. |
| **Dat** klopt. | That is correct. |
| Welke fiets? — **Deze**. | Which bicycle? — This one. |
| Welk huis? — **Dat**. | Which house? — That one. |

> *Dit* and *dat* are the default pronouns for unknown or unspecified things/people — equivalent to English *this is…* / *that is…*

---

## Choosing near vs far

- **Near (deze / dit)**: the noun is close to the speaker, or just introduced in conversation.
- **Far (die / dat)**: the noun is physically distant, or already known / previously mentioned.

In practice, Dutch speakers use *die/dat* very naturally for things already in shared awareness, even if physically close.

---

## Examples in context

| Dutch | English |
|-------|---------|
| **Dit** is mijn fiets en **die** is van Jan. | This is my bicycle and that one is Jan's. |
| Ik wil **deze** appel, niet **die**. | I want this apple, not that one. |
| **Dat** snap ik niet. | I do not understand that. |
| **Deze** week werk ik thuis. | This week I work from home. |

---

## Common mistakes

**1. Using *dit/dat* for *de*-words**

*Dit man is aardig.* ✗ → ***Deze** man is aardig.* ✓

**2. Using *deze/die* for singular *het*-words**

*Deze huis is groot.* ✗ → ***Dit** huis is groot.* ✓

**3. Forgetting that plurals use *deze/die***

*Dit kinderen* ✗ → ***Deze** kinderen* ✓  
*Dat fietsen* ✗ → ***Die** fietsen* ✓

---

## Quick reference

```
        Near        Far
de-word / plural:  deze       die
het-word singular: dit        dat
```
