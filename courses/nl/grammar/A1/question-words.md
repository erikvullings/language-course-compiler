# Question words

**CEFR level:** A1  
**Introduced in lesson:** 10  
**Depends on:** [Main clause word order](word-order.md)

---

## Explanation

Wh-questions (*vraagwoordzinnen*) ask for specific information rather than a yes/no answer. The question word occupies **position 1**, and the finite verb stays in **position 2** — the V2 rule still holds.

```
[Question word]  [Verb]  [Subject]  [Rest]  ?
 Waar             woont   jij?
 Where             lives   you?
```

---

## Dutch question words

| Dutch | English | Notes |
|-------|---------|-------|
| **wie** | who | *Wie is dat?* |
| **wat** | what | *Wat doe jij?* |
| **waar** | where | *Waar woon jij?* |
| **wanneer** | when | *Wanneer kom je?* |
| **hoe** | how | *Hoe gaat het?* |
| **waarom** | why | *Waarom lach je?* |
| **welk(e)** | which | *Welke trein neem je?* |
| **hoeveel** | how many / how much | *Hoeveel kost dat?* |
| **hoe lang** | how long | *Hoe lang duurt het?* |
| **hoe oud** | how old | *Hoe oud ben jij?* |

---

## Word order in wh-questions

The question word takes position 1. The finite verb follows in position 2. Subject comes in position 3.

| Position 1 | Position 2 | Position 3 | Rest |
|-----------|-----------|-----------|------|
| Wie | is | dat? | |
| Wat | doe | jij | hier? |
| Waar | woont | hij? | |
| Wanneer | komt | de trein? | |
| Hoe | heet | u? | |
| Waarom | lach | je? | |

> The *t*-drop rule applies whenever *jij/je* is the subject: *Wat doe **jij**?* (not *doet*).

---

## *wie* vs *wat*

- **wie** — refers to people
- **wat** — refers to things, ideas, actions

*Wie belt er?* — Who is calling?  
*Wat hoor je?* — What do you hear?

> After a preposition, *wie* becomes **wie** (for people) and *wat* is replaced by **waar + preposition** (for things):  
> *Met **wie** praat je?* — Who are you talking with?  
> *Waar **over** praat je?* / ***Waarover** praat je?* — What are you talking about?  
> *(This fused form is introduced at A2; at A1 you mainly need the simple forms.)*

## Common wh-question patterns

| Dutch | English |
|-------|---------|
| Hoe heet jij? | What is your name? *(lit. how are you called?)* |
| Hoe gaat het met jou? | How are you? |
| Waar kom jij vandaan? | Where are you from? |
| Hoeveel kost dit? | How much does this cost? |
| Welke bus gaat naar het centrum? | Which bus goes to the centre? |
| Hoe oud is hij? | How old is he? |
| Waarom huil je? | Why are you crying? |

## Common mistakes

**1. Keeping subject before verb after the question word**

*Waar jij woont?* ✗ → *Waar woon **jij**?* ✓

**2. Forgetting *t*-drop with *jij/je***

*Wat doe**t** jij?* ✗ → *Wat doe jij?* ✓

**3. Using *wat* for people**

*Wat is jouw naam?* is grammatically possible when asking about a thing (name), but *Hoe heet jij?* is the natural Dutch expression.

**4. Treating *hoe* compounds as two words**

*Hoe oud* and *hoe lang* are written as **two words** when used before a verb, but the combination *hoeveel* is written as **one word**.

## Quick reference

```
wie      →  who (people)
wat      →  what (things / ideas)
waar     →  where
wanneer  →  when
hoe      →  how  (also: hoe oud, hoe lang, hoe laat)
waarom   →  why
welk(e)  →  which
hoeveel  →  how many / how much

Order:  [Question word]  [Verb]  [Subject]  [Rest]  ?
```
