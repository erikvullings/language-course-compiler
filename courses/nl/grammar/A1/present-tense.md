# Present tense

**CEFR level:** A1  
**Introduced in lesson:** 2  
**Depends on:** [Subject pronouns](personal-pronouns.md)  
**Signal words:** *nu, vandaag, altijd, elke dag*

---

## Explanation

The Dutch present tense (*onvoltooid tegenwoordige tijd*) is used for:

- facts and habits: *Ik drink elke dag koffie.*
- actions happening right now: *Hij werkt nu.*
- scheduled future events: *De trein vertrekt om 10 uur.*

There is **no continuous tense** in Dutch (no equivalent of *I am working*). The simple present covers both.

---

## Forming the present tense

### Step 1 — find the stem

The stem is the infinitive minus the final **-en** (or **-n** if the infinitive ends in just **-n**).

| Infinitive | Stem |
|------------|------|
| werk**en** | werk |
| woon**en** | woon |
| spel**en** | spel |
| lez**en** | lees *(spelling rule: double vowel in open syllable)* |

> **Spelling rule:** when removing *-en* leaves a single vowel in an open syllable, double that vowel to preserve the long sound.  
> *lezen* → stem *lez-* → but written *lees* to keep the long *ee* sound.

### Step 2 — add endings

| Subject | Ending | Example (werken) |
|---------|--------|-----------------|
| ik | stem only | ik **werk** |
| jij/je | stem + **t** | jij **werkt** |
| u | stem + **t** | u **werkt** |
| hij/zij/het | stem + **t** | hij **werkt** |
| wij/we | infinitive | wij **werken** |
| jullie | infinitive | jullie **werken** |
| zij/ze (pl.) | infinitive | zij **werken** |

> **Rule of thumb:** *ik* = bare stem; everything else singular = stem + *t*; plural = infinitive.

---

## The *t*-drop rule (very important)

When **jij** or **je** comes **after** the verb (inversion), the *-t* is dropped:

| Normal order | Inverted (question / fronted adverb) |
|--------------|--------------------------------------|
| Jij werk**t** thuis. | Werk jij thuis? |
| Je werk**t** veel. | Vandaag werk je veel. |

This rule applies **only** to *jij/je*, never to *u*, *hij*, *zij*, or *het*.

## Examples

| Dutch | English |
|-------|---------|
| Ik **werk** elke dag. | I work every day. |
| Jij **werkt** thuis. | You work from home. |
| Hij **woont** in Utrecht. | He lives in Utrecht. |
| Wij **spreken** Nederlands. | We speak Dutch. |
| **Werk** jij hier? | Do you work here? *(t-drop)* |
| Vandaag **speel** ik voetbal. | Today I play football. |
| Ze **lezen** altijd 's avonds. | They always read in the evening. |

## Common mistakes

**1. Adding -t to *ik***

*ik werkt* ✗ → *ik werk* ✓

**2. Forgetting the *t*-drop**

*Werk**t** jij hier?* ✗ → *Werk jij hier?* ✓

**3. Spelling the stem incorrectly**

Remove *-en*, then check whether you need to double the vowel or drop a double consonant.

| Infinitive | Stem | Pitfall |
|------------|------|---------|
| stoppen | stop | *stopt* (not *stoppt*) |
| lezen | lees | long vowel must be preserved |
| rijden | rijd | *rijdt* (the *d* stays) |

**4. Using -t with plurals**

Plurals always use the full infinitive, never *-t*.

---

## Quick reference

```
ik      → stem
jij/je  → stem + t   (drops in inversion)
u       → stem + t   (never drops)
hij/zij/het → stem + t
wij/we/jullie/zij → infinitive
```
